## Technologies & Tools

### Front-end:

- html
- bootstrap

### Backend:

- php
- mySql
- composer (for manage dependencys)

## Features

- create product
- view all products
- update product
- search for products
- delete product

## Usage

### Install Dependencies (frontend & backend)

```
composer install
```

### Run

```
cd ./public
# Run on port :8080
php -S localhost:8080
```
